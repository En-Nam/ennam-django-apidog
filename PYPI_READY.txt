================================================================================
TROUBLESHOOTING
================================================================================

If GitHub Actions Publish Fails:

  1. Check the workflow logs:
     - Go to Actions tab
     - Click failed workflow
     - Review error messages

  2. Common issues:
     - PyPI credentials not configured (requires Trusted Publishers setup)
     - Version already exists on PyPI (choose different version)
     - Build fails due to code issues (fix and re-push tag)

  3. Recovery:
     - Fix the issue
     - Delete git tag
     - Create new tag
     - Push tag
     - Create new GitHub Release

If Package Not Appearing on PyPI:

  1. Wait 5-10 minutes for PyPI indexing

  2. Clear pip cache:
     pip cache purge

  3. Try installing again:
     pip install --no-cache-dir ennam-django-apidog

See PUBLISH.md for more troubleshooting.
